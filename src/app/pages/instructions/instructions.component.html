LEDs (Lights)

Vertigo has a set of Light Emitting Diodes (LEDs) which indicate information about the current status of Vertigo. There are three LEDs on the device. These are Green, Orange and Blue in colour.

Turning vertigo on

Vertigo has a slide switch on the side of the case which turns it on. Slide it towards the front of the unit to turn on. The unit will flash a green led on the top when it is turned on and hasn't got a GPS position. If plugged in to usb, the switch selects whether Vertigo is powered by USB or Battery.

Starting and stopping logging

Before starting logging, make sure that and SD card has been inserted and, if GPS is needed for the experiment, that a GPS position has been found. Vertigo will stop flashing the green LED once it has found it's GPS position and the green light will stay on solid green.

When Vertigo is ready, press the LOG button to begin logging. Note that orange LED lights up solidly indicating that Vertigo is logging.

During logging, press the LOG button again to stop. Vertigo can now be powered off, or another logging cycle can begin.

Clear and format SD Card
Formatting

Before Vertigo can use the SD card, it must be formatted correctly to be used. Vertigo comes with an SD card, however if it is in the wrong format or you are using a different SD card, you need to format it as FAT32 or exFAT. Follow the instructions here to format it: https://www.lifewire.com/format-sd-card-using-windows-4128719

Clearing

Once formatted, Vertigo will write calibration and default configuration files to the SD card. If you want to remove any old log files from the SD card, or if you run out of space. Follow the instrutions below to clear the SD card:

Whilst vertigo is turned off, press and hold the log button. Turn on vertigo whilst continuing to hold down the button. The orange LED will flash rapidly whilst the SD card is clearing.

Inserting and removing SD cards

The SD card slot it spring loaded, to insert an SD card, push it until you hear a click and then release. To remove the SD card, use something to push it again until it clicks and then release, the SD card should spring out far enough to pull out.

Charging
Charging Vertigo is super easy since the battery charger is built in! You just need a mini-USB cable, and follow these instructions to make sure your Vertigo is fully charged. 1. Make sure Vertigo has its battery connected and is powered OFF. 2. Connect the mini-USB connector to the USB port. 3. The green CHARGE LED can be seen when looking through the power switch next to the USB port. 4. When the green CHARGE LED is extinguished, the battery is charged.
